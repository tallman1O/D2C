## üöÄ Project Overview

**D2C** (Design to Code) is a modern web application that allows users to upload wireframes, generate code from designs, and manage their projects efficiently. It leverages AI and a robust tech stack to streamline the design-to-development workflow.

---

## ‚ú® Features

- üîí **Authentication**: Secure login and user management.
- üñºÔ∏è **Wireframe Upload**: Upload and store wireframe images.
- ü§ñ **AI Model Integration**: Generate code from wireframes using AI.
- üìù **User Prompts**: Customize code generation with user input.
- üì¶ **Project Dashboard**: Manage and view all your wireframes and generated code.
- ‚ö° **Real-time Updates**: Instant feedback and updates on your projects.

## üßë‚Äçüíª Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/D2C.git
   cd D2C
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   - Copy `.env.example` to `.env` and fill in your credentials.

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open [http://localhost:3000](http://localhost:3000) to view it in your browser.**
